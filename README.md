# desafio-04-formulario-mentoria

Este é um projeto desenvolvido durante um desafio da Rocketseat para praticar formulários e boas práticas de desenvolvimento. O objetivo foi criar um formulário intermediário seguindo um layout fornecido, com foco em validação de dados, acessibilidade e usabilidade.
